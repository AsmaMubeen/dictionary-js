# Usage
<!-- usage -->
```sh-session
$ npm install -g dictionary-js
$ dict COMMAND
running command...
$ dict (-v|--version|version)
dictionary-js/0.0.0 win32-x64 node-v12.13.0
$ dict --help [COMMAND]
USAGE
  $ dict COMMAND
...
```

```
OPTIONS
  --all  see all commands in CLI
```
